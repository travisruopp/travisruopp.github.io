Pet Match Maker

Project Description

A project designed to revolutionize the way pets and owners find their perfect match, a comprehensive pet matchmaking program driven by Java and backed by a robust database. We leveraged both front-end and back-end technologies to create a seamless user experience while ensuring efficient data management and retrieval. Through an intuitive user interface built with Thymeleaf and Spring Boot, pet owners can search for compatible companions based on a creative quiz. Behind the scenes, we implemented a Java-driven database, utilizing frameworks to persist pet profiles and user preferences. This ensures fast and reliable access to data, facilitating real-time matchmaking and fostering meaningful connections between pets and their future owners.

Skills Used:

Java, HTML, CSS, MVC, OOP, Project Management, Alpha Testing
